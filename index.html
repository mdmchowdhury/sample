base tag

W3Schools

Note:The base tag specifies the base URL/target for all relative URLs in a document.

There can be at maximum one base element in a document, and it must be inside the head element. Notice that we have only specified a relative address for the image. Since we have specified a base URL in the head section, the browser will look for the image at "https://www.w3schools.com/images/stickman.gif".
Notice that the link opens in a new window, even if it has no target="_blank" attribute. This is because the target attribute of the base element is set to "_blank".
